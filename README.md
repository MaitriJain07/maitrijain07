## 🧠 About Me

I'm a **Computer Science undergrad** doing a dual degree — B.Tech + IIT Madras BS in Data Science — and I love turning half-baked ideas into real, working web apps.

From building an **NGO library system** from scratch to shipping a **trekking management platform**, I'm the person who jumps in as sole developer and figures it out. - 🌱 **Learning:** Linux internals, container basics & system design
