## 🧠 About Me

I'm a **Computer Science undergrad** doing a dual degree — B.Tech + IIT Madras BS in Data Science. I build things that **actually ship**.

I started with Flask. Then I containerized with Docker. Then I deployed on AWS. Now I'm breaking monoliths into microservices, automating infrastructure, and learning to monitor what I build.

I don't just write code — I **own the stack**. From the first line of Python to the live URL, I make it work.

- 🔭 **Currently:** Building a microservices version of my live NGO app (auth, books, recommendations)
- 🌱 **Learning:** Linux internals, Docker, AWS, Terraform, CI/CD (GitHub Actions), Prometheus
- 🏆 **3rd place** — IIT Madras Cybersecurity CTF (steganography, network analysis, backend vulns)
- ⚡ **Fun fact:** I debug better with lo-fi music and I write production code at 2 AM

> *"Build it. Break it. Fix it. Ship it. Repeat."*
